# Capstone Project Ideas

Choose a project that matches your skill level and interests. Each project integrates concepts from all previous chapters.

Consider:
- **Time available**: 1-12 weeks
- **Hardware access**: Simulation only or real robot?
- **Skill level**: Current experience
- **Interest**: What excites you?

## ‚≠ê‚≠ê‚òÜ‚òÜ‚òÜ Beginner Projects

### 1. Balance Controller (1-2 weeks)

**Goal**: Keep robot upright using IMU feedback

**Requirements**:
- Read IMU data
- Implement PID controller
- Control ankle/hip joints
- Test with disturbances

**Learning outcomes**: Reactive control, sensor integration

---

### 2. Object Tracking (1-2 weeks)

**Goal**: Track colored object with camera

**Requirements**:
- Camera image processing
- Detect colored object
- Control head/camera to follow
- Maintain object in center of view

**Learning outcomes**: Vision processing, servo control

---

## ‚≠ê‚≠ê‚≠ê‚òÜ‚òÜ Intermediate Projects

### 3. Autonomous Navigation (3-4 weeks)

**Goal**: Navigate to goal while avoiding obstacles

**Requirements**:
- A* path planning
- LIDAR-based obstacle detection
- Reactive obstacle avoidance
- Reach goal position

**Learning outcomes**: Planning + reactive control

---

### 4. Object Manipulation (3-4 weeks)

**Goal**: Pick up object and place it

**Requirements**:
- Vision-based object detection
- Inverse kinematics for reaching
- Grasp force control
- Place object at target

**Learning outcomes**: Vision, IK, force control

---

## ‚≠ê‚≠ê‚≠ê‚≠ê‚òÜ Advanced Projects

### 5. Bipedal Walking (6-8 weeks)

**Goal**: Implement stable walking gait

**Requirements**:
- Zero-moment point (ZMP) planning
- Footstep planning
- Balance control
- Walk forward/turn

**Learning outcomes**: Locomotion, advanced control

---

### 6. Dynamic Reaching (6-8 weeks)

**Goal**: Catch moving objects

**Requirements**:
- Track moving object
- Predict trajectory
- Plan interception
- Time grasp correctly

**Learning outcomes**: Prediction, optimization

---

## ‚≠ê‚≠ê‚≠ê‚≠ê‚≠ê Expert Projects

### 7. Reinforcement Learning Locomotion (8-12 weeks)

**Goal**: Learn walking policy with RL

**Requirements**:
- Implement PPO/SAC
- Train in simulation
- Domain randomization
- Transfer to real robot

**Learning outcomes**: Deep RL, sim-to-real

---

### 8. Multi-Robot Coordination (8-10 weeks)

**Goal**: Coordinate multiple robots for task

**Requirements**:
- Multi-agent planning
- Communication protocol
- Collision avoidance
- Task allocation

**Learning outcomes**: Distributed systems
